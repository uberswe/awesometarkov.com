## Item Databases & Flea Market

Real-time flea market prices, item databases, and market analysis tools.

- [Tarkov Market](https://tarkov-market.com) - Most comprehensive flea market monitoring with price history charts
- [Tarkov.dev](https://tarkov.dev) - Open-source alternative with real-time prices and free GraphQL API
- [Rat Scanner](https://ratscanner.com) - Gold standard OCR tool for in-game price checking
- [Tarkov Bazaar](https://www.overwolf.com/app/Tarkov_Bazaar) - Overwolf overlay for hotkey price checking
- [Tarkov Guru](https://tarkov.guru) - Barter trade profitability analysis
- [Blitz.gg EFT](https://blitz.gg/eft) - Market trend insights and player stats

## Interactive Maps

Navigate Tarkov's complex maps with these interactive tools.

- [MapGenie](https://mapgenie.io/tarkov) - Polished interactive maps with mobile companion app
- [Tarkov.dev Maps](https://tarkov.dev/maps) - 2D and 3D views with quest markers
- [RE3MR](https://reemr.se) - High-quality isometric 3D maps
- [TarkovQuestie](https://tarkovquestie.com) - Live position tracking via screenshot analysis
- [GameMaps.net](https://gamemaps.net/tarkov) - Community-created map variants
